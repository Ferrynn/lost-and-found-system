# Lost & Found System

A simple Lost & Found web page I built while learning HTML and CSS.

It lets users view lost items and report new ones through a basic form. Nothing fancy—just a practice project to understand how web pages are structured.

---

## What I built

- A basic homepage layout  
- A list section for lost items  
- A simple report form  
- Basic navigation between sections  

---

## Tech used

- HTML  
- CSS  

---

## Why I made this

I wanted to practice building real-looking pages instead of just small code exercises. This helped me understand layouts, forms, and structuring a simple web project.
